# Agent Entry Point

Before changing code in this repository, read this file first.

Required reading:

- Backend work: read `packages/backend/convention.txt`.
- Frontend work: read `packages/frontend/convention.txt`.
- Cross-package work: read both convention files.

Repository conventions:

- Follow the nearest package convention file.
- Keep file and folder names in kebab-case.
- Do not introduce unrelated refactors while implementing a task.
- Prefer existing project patterns over new abstractions.
